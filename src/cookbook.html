Dealing with reported OSM issues

DWG Helper panel this guide refers to is located either in a browser sidebar, or in an extension popup.

One issue that is not worthy of creating a ticket

These are typically reported by only one user.

  1. Open the issue page (https://www.openstreetmap.org/issues/issueId)
  2. Do whatever actions required to solve the issue, or what is possible to do before contacting the reporting user.
  3. Contact the reporting user by selecting Quick message reporting user... in DWG Helper panel. This opens an OSM message form (https://www.openstreetmap.org/message/new/username) with the issue report copied.
  4. Write a message and send it.
  5. Press the resolve issue button in the Ongoing actions section of the panel to resolve the issue.
